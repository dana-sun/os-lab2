
To Run:

1. Run the lab2.cpp on linserv
2. In order to run, type: g++ lab2.cpp -std=c++11
3. Type ./a.out
4. User will be prompted to first enter the number of processes (for example: 2)
5. User will be prompted to enter either 1 or 2 depending on if they want the FIFO or RR algorithm
6. User will be asked to enter the phases for the first process (for example: 3 2 3 2)
7. User will keep being asked to enter the number of phases depending on how many processes are being asked for
